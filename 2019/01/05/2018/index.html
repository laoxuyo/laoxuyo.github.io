2018小事记

发表于 2019-01-05 | 分类于 总结

这不是年度总结，这是长篇大论，内容来自朋友圈、笔记、代码提交记录以及回忆。

一月

第一次有人帮我买蛋糕陪我过生日，而且是全公司的人。记得回到家之后马哥喝醉了给我打电话说：“我很少遇到像你这样刚毕业的女孩子，你天真、善良、没有心机，……”，我很感动，哭着听完了，但是我现在想想我你是不是在说我傻。

这是我待的第一家公司，当时只有一个前端，虽然马哥是个iOS开发，但是也帮我解决了不少问题。

二月

过年回家跟我哥闹得不愉快，几个月之后他问我在杭州过得好吗，然后我就哭了。

三月

感觉公司在悄悄地改变，变得很压抑，一些同事也走了。加了几天班之后变得空闲起来，当时有辞职的打算，但是考虑到自己的能力，就暂时没走。

四月

4月5日，第一次去现场看电竞比赛。

4月27日，老板毫无预兆地宣布公司倒闭！给了我们两个选择，第一个是跟着他去新公司，第二是自己去找工作。问了同事，说这家公司在xxx园区，一般跟政府有点关系才能进去，应该比较稳定。（现在我只想说：我信你个鬼,你个糟老头子坏得很。）还有一个原因就是技术特别菜，不敢出去找工作。然后就这样决定跟着老板去了新公司。

4月28日，分期买了个Mac Pro。

4月29日，去看了一场演唱会

4月30日，南浔。

五月

5月份开始，真正开始学习前端。新公司没什么事情做，然后就在群里聊天，幸好这个群不是什么不正经的群，很多时候都有下午茶（每天下午一道面试题），群里还有大佬很详细地解答，对于我这种菜鸟来说非常地友好。

其实最最主要的是认识了老王。当时建了一个玩游戏的四人小群，老王就经常在里面给我们开小灶，幸福。

我现在还保存着比较重要的截图和群里大佬的语录，这些我自己看看就好，不敢放出来。

六月

儿童节那天，老王一人送了一本书。

虽然我做不到这样子，但是从那天开始接下来的几个月，真的有每天学习一小会。

可能老王都不记得他说过这样的话了，可爱，哈哈哈。

六月中旬，感受了一次荒岛生活。

六月底，老王提出每周写周报，不限于生活学习。这个作用真的很大，至少我会每周都想着学些什么，要怎么总结才好。虽然说不该为了什么才去学习，但是没有一点约束我是没定力去学习的。

七月

无所事事

八月

这个时候应该跟着老王写开源项目了，学到了超级多东西。比如git，比如一个小时入门react。
记得老王写rc-bmap的时候曾经用路飞当背景，他问我们知道为什么吗，然后他说：“不是因为我喜欢海贼王，只是我很喜欢路飞的那种精神。不是打，是在这路上，你会遇到各种各样的人，各种各样的问题，重点是不要逃避，要学会面对，想办法解决。做我们这行，在写一个东西，不在于你写的有多么牛逼，而在于你做这个东西的时候，想了什么，学到了什么，还有没有什么可以做的更好的地方。这几点都达到了，你的水平自然也就上来了。总想着xxxx时候再xxxx吧，这他吗是程序猿最大的谎言，等到后面，你真的还会保持同样的热情继续下去吗？我是做不到。所以我就在搞一个东西的时候，投入全部的热情，把该搞懂的搞懂，该学会的学会，别给自己留尾巴。”

九月

club变成了胡扯蛋群，我一直认为有我的原因，很多时候我担任的是插科打诨的角色。看着这个讽刺的群名之后一度想自闭。

期间在老王的指导下还写了一点代码，看了一些代码，老王一针见血地指出我的缺点：不沟通不沟通不沟通。

十月

因为新公司每个月都拖延工资（现在一些同事正在劳动仲裁，准确地说我还有半个月工资没拿回来），所以我下定决心重新找工作。在杭州面试了几天，突然就决定去深圳了，我也不知道为什么。

最不舍的就是杭州的朋友了，有可能一辈子都见不到了，21号检票的时候，泣不成声，其实我一直以为我挺冷漠的。

然后就是深圳广州的公司都去面试，最后决定留在汤臣倍健，因为实在是不想折腾了，也不想成为“公司倒闭·姬”。

十一月十二月

正如老王所说，我开始放飞自我，堕落了。。。每天五点下班就是玩游戏看小说

其实我是在迷茫，我不知道还要不要在这条路上走下去。

2019

目标就是赚钱了，既然迷茫，不知道做什么，为什么不把当前的事情做好。我有一段时间特别喜欢看重生小说，我觉得是对过去不满意，我想回到过去改变，我现在就在想啊，对于未来的你来说，现在不就是过去吗？
